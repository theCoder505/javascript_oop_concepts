


### OOP ==> INHERITANCE | POLYMORPHISM | ENCAPSULATION | ABSTRACTION 
#               ↓               ↓               ↓              ↓
#               ↓               ↓               ↓              ↓
#               ↓               ↓               ↓              ↓
#           Reusability |   Flexibility  |   Security  |  Simplicity
#
# Inheritance → Reusability — write once, use many times 
# Polymorphism → Flexibility — same method, different behaviors 
# Encapsulation → Security — protect and hide data 
# Abstraction → Simplicity — hide complexity, show only what matters 
#
# Inheritance = বাবার গুণ ছেলে পাবে ♻️ (উত্তরাধিকার)
# Polymorphism = এক মানুষ, অনেক রূপ 🎭 (বহুরূপিতা)
# Encapsulation = তালা দিয়ে রাখা 🔒 (আবদ্ধকরণ)
# Abstraction = শুধু দরকারি জিনিস দেখানো 🎯 (বিমূর্তকরণ)
